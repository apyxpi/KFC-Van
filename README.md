# KFC-Van
KFC Van retuxture [FiveM Ready]. KFC (short for Kentucky Fried Chicken) is an American fast food restaurant chain headquartered in Louisville, Kentucky, that specializes in fried chicken.

# ⌞Installation⌝

*1) Download the file.*

*2) Pull out the folder and paste it into the server.*

*3) Type “start kfc” in your “server.cfg”.*
